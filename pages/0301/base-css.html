cmsYou should usually important text classify a div using multiple column classes so it behaves differently depending on the screen size (this is the heart emphasize text of what makes bootstrap responsive). eg: a div with classes col-xs-6 and col-sm-4 will span half the screen on mobile phone (xs) and 1/3 of the screen on tablets(sm)

Each set of nested divs also span up to 12 columns of their parent div. NOTE: Since each .col class has 15px padding on either side, you should usually wrap nested columns in a .row, which has -15px margins. This avoids duplicating the padding, and keeps the content lined up between nested and non-nested col classes.
